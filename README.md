# VehicleTrack
This project is a simple implementation of Object-Oriented Programming (OOPS) concepts in C++. It was developed as part of an academic assignment.

## Description
The system is designed to manage different types of vehicles such as Gas Cars and Electric Vehicles. It stores basic details like make, model, year, and fuel/battery capacity, and displays them in a structured format.

## OOPS Concepts Used
- Inheritance  
- Encapsulation  
- Abstraction  

## Features
- Separate classes for different vehicle types  
- Constructor-based initialization  
- Clear and simple output using display functions
